## Установка зависимостей
Чтобы установить необходимые зависимости, выполните следующие шаги:

1. Скачайте репозиторий и перейдите в каталог проекта:
git clone https:/github.com/Mafun-gpu/OS_Kursovaya_Frolov.git
cd OS_Kursovaya_Frolov.git

2. Сделайте скрипт 'install.sh' исполняемым:
chmod +x install.sh

3. Запустите скрипт установки:
./install.sh

После усешной установки вы сможете запустить файл скрипта программы script.py.

Для тестирвоания программы следует изменять содержимое файла jobs.json.

!!! Что бы запустить программу, в консоль linux следует ввести следующую комманду:
    python script.py jobs.json
